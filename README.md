This project is a powerful automation tool designed to fill out Google Forms automatically using a combination of Python and AutoHotkey (AHK). It streamlines repetitive data entry tasks, saving time and reducing manual errors.
✨ Features
Dual-Engine Automation: Leverages Python for logic and AutoHotkey for seamless Windows-level interaction.
Auto-Submit: Includes a dedicated Python script (autosubmit.py) to handle form completions.
Smart Paste: Uses AutoPaste_Manual-Auto.ahk for efficient data handling.
User-Friendly: Designed to be lightweight and easy to trigger.
🛠️ Built With
Python - For backend logic and submission handling.
AutoHotkey - For GUI automation and keystroke simulation.
🚀 Getting Started
Install Prerequisites:
Install Python 3.x.
Install AutoHotkey.
Clone the Repository:
bash
git clone https://github.com
Use code with caution.
Run the Script:
Launch the .ahk files for keyboard automation.
Execute autosubmit.py to manage the form submission process.
📂 File Structure
FormFill_v2.ahk: The core automation script.
autosubmit.py: Handles the final form submission.
AutoPaste_Manual-Auto.ahk: Manages data pasting logic.
po.ahk: Supporting automation script.
